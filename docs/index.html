Matchmakeアプリのプライバシーポリシー

Matchmakeアプリ（以下、「本アプリ」といいます）をご利用いただき、ありがとうございます。本アプリは、お客様のプライバシーを尊重し、個人情報の取り扱いに細心の注意を払っています。本プライバシーポリシーでは、本アプリが収集する情報について説明いたします。

1. 収集する情報

本アプリは、以下の情報を収集します。

1.1 端末情報

本アプリは、お使いの端末の種類や識別子などの情報を収集します。

1.2 OSバージョン

本アプリは、お使いの端末のオペレーティングシステム（OS）のバージョン情報を収集します。

1.3 クラッシュ時の原因

本アプリは、クラッシュした場合には、その原因や詳細情報を収集することがあります。これにより、本アプリの安定性と品質向上に役立てることができます。

2. 取得した情報の利用

本アプリは、収集した情報を以下の目的で利用することがあります。

  • 本アプリの提供、運営、保護、および改善のため
  • お客様へのサポートやお問い合わせへの対応のため
  • 本アプリの機能やサービスの向上のため
  • お客様への情報提供やマーケティング目的のため（匿名化された形式でのみ）

3. 情報の共有

本アプリは、法的な要求に基づいて、お客様の情報を第三者と共有することがあります。

4. セキュリティ

本アプリは、お客様の情報を適切に保護するために、適切な物理的、電子的、および組織的な手段を講じます。
